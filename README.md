# Atividade prática de RAG

## Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/Wooo589/pipeline-RAG.git
cd pipeline-RAG
```
### 2. Crie um ambiente virtual

Linux/macOS:
```bash
python3 -m venv venv
source venv/bin/activate
```
Windows:
```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Instale o arquivo requirements.txt
```bash
pip install -r requirements.txt
```
## Execução
Ambas as alternativas são válidas, irá depender da sua máquina:
```bash
python3 src/rag.py
```
```bash
python src/rag.py
```
